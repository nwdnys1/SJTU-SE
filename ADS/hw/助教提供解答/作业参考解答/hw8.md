## KMP
（对于字符串P，其next数组定义按课件：∀j≥1, N(P,j) = {0≤t<j | P[0,t) = P[j-t,j)}, next[j] = max{N(P, j)}, next[0] = -1 ）

字符串 “ABCDAABCDABCG” 对应的next表为 \[-1, 0, 0, 0, 0, 1, 1, 2, 3, 4, 5, 2, 3\] 。
